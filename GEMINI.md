# Project: lophone

## Tech Stack
- **Frontend Framework**: React 19
- **Build Tool**: Vite 8 (Beta)
- **Mobile Framework**: Capacitor 8 (Targeting Android)
- **Styling**: Tailwind CSS 3
- **Routing**: React Router 7
- **Icons**: Lucide React

## Project Goals
- Turn an old mobile phone into an anti-theft device.

## AI Instructions
- Focus on mobile-first, responsive UI using Tailwind CSS.
- When adding features, consider Capacitor plugin compatibility for hardware access (camera, sensors, etc.).
- Maintain strict type safety and modern React patterns.
