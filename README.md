# Samsung Anti-Theft Tracker Web App (Ghost-Hardened)

A standalone, React-based web application containing an interactive, mathematically optimized checklist for repurposing old Samsung devices into hyper-stealthy, off-grid anti-theft car tracking beacons.

## Current Features
*   **Dual-Network Beaconing:** Leverages both **Samsung Find** and **Google Find My Device** (with "Everywhere" network settings) for maximum recovery reach.
*   **Ghost Hardening:** Granular steps for disabling system alerts, silencing tracking notifications, and stripping the UI to zero-visibility.
*   **Dynamic SIM Toggles:** Seamlessly switch between "No SIM (Offline)" and "With SIM (Full Features)" modes.
*   **Geofence Tripwires:** Intelligent "Modes & Routines" for "Home" and "Work" that automatically kill Wi-Fi and engage power-saving the moment a vehicle is moved.
*   **Expert Data Security:** Guidance on "Account Isolation," "Contact Handshake scrubbing," and "Death Gasp" location pings.
*   **Hardware Protection:** Strategies for battery longevity and thermal safety in permanent vehicle-mount scenarios.
